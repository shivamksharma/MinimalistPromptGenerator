## üöÄ Overview

**Minimalist Prompt Generator** is a sleek, user-friendly shell prompt builder designed for creating personalized PS1 prompts effortlessly. Focused on simplicity, it provides a clean interface for bashrc customization and zsh prompt customization, enhancing your Linux terminal customization with a minimalist aesthetic.

Inspired by the functionality and simplicity of Bashrc Generator.

## üí° Why This Tool?

Fast, animation-free, hyper-minimal drag-and-drop PS1 generator for bash prompt generator. Tailored for developers seeking speed, simplicity, and distraction-free shell prompt builder. No bloat, pure functionality for crafting perfect terminal prompts in Linux terminal customization.

## ‚ú® Features

- **Visual Editor**: Drag-and-drop elements to design your PS1 generator prompt.
- **Real-time Preview**: See your bash prompt generator changes instantly.
- **Syntax Customization**: Adjust colors, styles, and layout for bashrc customization and zsh prompt customization.
- **Export Options**: Save your custom shell prompt builder as text, HTML, or Markdown.
- **Save & Load Configurations**: Easily store and retrieve custom configurations for Linux terminal customization.
- **Help & Documentation**: Quick syntax guide for PS1 elements, ANSI color codes, keyboard shortcuts, and feedback support.

## üîß Installation

Clone the repository:

```bash
git clone https://github.com/shivamksharma/MinimalistPromptGenerator.git
```

Navigate into the project folder:

```bash
cd MinimalistPromptGenerator
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
